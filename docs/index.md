# 欢迎来到 **DSEG 知识库**！
这是一个部署在DSEG-NAS上的知识库，愿景是成为一个知识共享的平台，搜集总结与研究工作相关的知识，方便大家快速进入感兴趣的领域（参见：[掌握概念而不是拘泥细节](intro/vision.md)）。

Please enjoy yourself~😊

本平台具备以下功能：

 - 集成了留言系统，通过Github账号登陆并授予权限后，可以在相关页面下参与讨论。
 - 集成一系列Markdown插件，支持提示框和代码选项卡等高级功能。
